    + `consent`: agreement, permission, or approval; age of consent
    + `conserve`: 1.节省,节约；2.保护,保存；3.蜜饯,果酱

    + `conscience`: 良知,良心,内疚
        - `subconscious`: n.潜意识，下意识； adj. 下意识的，潜意识的.
    + `considerable`: 相当大的,重要的,可观的

    + `constant`: continuous, unchanging, faithful: 经常的,不变的,坚定的,常量
    + `constrain`: 约束,限制,强迫
    + `contempt`: 轻视,鄙视

    + `contradict`: 反驳，与...发生矛盾，抵触

    + `obligate`: v. 强迫， 迫使
    + `obligation`:ob向着 + lig绑住 = 强制，责任
    + `religious`: 绑回神身边，信仰.
    + `allied`: adj. 盟国的，盟军的。 al向着 + lied捆绑
    + `alliance`:n. 联盟，同盟 al向着 + lia捆绑

* **["labor"]** 表示劳动

    + `collaboration`: n. 合作，协作
        - `collaborative`: adj. 合作的，协作的
    + `laborious`: adj. 耗时费力的，辛苦的，艰难的

    + `propose`: 提出,建议,计划,打算

* **["pre..."]** begin with "pre"
    + `preliminary`: **[prɪˈlɪmɪneri]** adj. 初步的,预备的
    + `prejudice`: n. 偏见,成见,损害,侵害. v. 对...有成见，侵害，损害


* **["pet"]** = seek, 表示追求,寻求
    + `competitive`: com一起 + pet追逐 = 共同追逐某物 = 竞争的,角逐的

    + `petulant`: pet + ulnat多...的 = 追求太多导致脾气不好 = 大发脾气的,任性的
    + `impetus`: im在...内 + pet = 里面有追求=冲动 推动,促进,推动力
    + `incompetent`: in无 + competent有能力的 = 无能力的,不称职的,不合格的

    + `derive`: 起源,衍生；取得
        - `deduction`: 推导

        - induce v. 劝诱,诱导,引导
    - `deprive`: "de-private" 剥夺,丧失,使不能拥有
 
* **["re..."]**
    + `reproach`: 责备,指责
    + `refine`: 提炼,精炼,改进
        - `refinement`: 举止文雅,有教养。

* **["reg"]** 拉丁： 表示引导，统治，王权
